On this section of the website you will be able to take a look at all the items that have showed up on any of the metal gear series

Game Name Description
Metal Gear Antenna Allows the user to send and recieve communications
Metal Gear Binoculars Allows user to surveillance the surrounding area
Metal Gear Body Armor Halves the damage caused by enemy weapons
Metal Gear Bomb Blast Suit Protects the user from strong winds and explosive blasts
Metal Gear Card Opens security doors based on security level on the card and door
Metal Gear Cardboard Box Used to hide from cameras and enemies
Metal Gear Cigarettes Affects the users health but does allow the user to see location of infrared sensors
Metal Gear Compass Gives the user directions
Metal Gear Flashlight Illuminates underground passages
Metal Gear Gas Mask Required for survival of gassed areas
Metal Gear Infrared Goggles Used to detect infrared alarm sensors
Metal Gear Mine Detector Identifies the location of enemy mines
Metal Gear O2 Cylinder Allows the user to breathe underwater
Metal Gear Parachute The user is able to descend from elevated heights
Metal Gear Ration replenishes the users health
Metal Gear Silencer reduces the noise pollution caused by a fire arm. Allows snake to not be spotted by enemies easily.
Metal Gear Enemy Uniform Used to bypass enemy checkpoints
Metal Gear Ammunition Cache Contains a flethura of ammunition that refills snakes inventory full.
